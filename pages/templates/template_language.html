Template Language

1. 반복문

{% for menu in menus %}

{{ menu }}

{% endfor %}
{% comment %} {{forloop}} DTL에서 for문 안에서 자동으로 생기는 객체 {% endcomment %} {% for menu in menus %}

{{ forloop.counter }}. {{menu}}

{% endfor %}
{% for user in empty_list %}

{{ user }}

{% empty %}

현재 가입한 유저가 없습니다.

{% endfor %}

2. 조건문

{% if '짜장면' in menus %}

짜장면엔 고춧가루지!

{% endif %}
{% for menu in menus %}

{{ forloop.counter }}번째 도는중....

{% if forloop.first %}

{{ menu }} + 고춧가루

{% else %}

{{ menu }}

{% endif %} {% endfor %}

3. length filter 활용

{% for message in messages %} {% if message|length > 5 %}

{{message}}, 글자가 너무 길어요.

{% else %}

{{message}}, {{message|length }}

{% endif %} {% endfor %}

5. 글자 수 제한(truncate)

{{my_sentence|truncatewords:3}}

{% comment %} 단어 단위로 제한 {% endcomment %}

{{my_sentence|truncatechars:3}}

{% comment %} 문자 단위로 제한 (3번째는 포함X) {% endcomment %}

{{my_sentence|truncatechars:10}}

{% comment %} 문자 단위로 제한 (10번째는 포함X) {% endcomment %}
